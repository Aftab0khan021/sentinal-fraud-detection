# SentinAL Fraud Detection System

Welcome to the documentation for SentinAL, an enterprise-grade fraud detection system using Graph Neural Networks.

## Features

- **Graph Neural Networks:** R-GCN based fraud detection.
- **Explainable AI:** Natural language explanations for flagged users.
- **Advanced Operations:** Containerized, Monitored, and Documented.
- **Compliance:** Secure Audit Logging and GDPR readiness.

## Getting Started

Check out the [Setup Guide](setup.md) to get started.

## Architecture

The system consists of:
1. **Frontend:** React + Vite
2. **Backend:** FastAPI + PyTorch
3. **Database:** In-memory NetworkX (Simulated) / PyG
4. **Monitoring:** Prometheus + Grafana
